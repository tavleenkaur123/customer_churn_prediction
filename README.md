**CUSTOMER CHURN PREDICTION**
Customer churn is a critical concern for telecom companies, as losing customers directly impacts revenue and growth. This project focuses on building a machine learning model to predict customer churn using historical telecom customer data. By identifying high-risk customers, businesses can take proactive steps to improve retention and optimize marketing strategies.

**Records**: ~7,000
**Features**: Demographics: gender, SeniorCitizen, Partner, Dependents
**Account info**: tenure, Contract, PaymentMethod, MonthlyCharges, TotalCharges
**Services subscribed**: PhoneService, InternetService, OnlineSecurity, StreamingTV, etc.
**Target Variable**: Churn (Yes/No)

**Project Objective**
1.Predict whether a customer is likely to churn.
2.Analyze patterns and trends that contribute to churn.
3.Deliver insights to help telecom providers retain customers.
